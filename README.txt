# Obtaining the Code

You can obtain the code here: https://github.com/psiionik/OMSCSMachineLearning_Project4Code

These are all Jupyter Notebook files so you should be able to setup your environment for all the necessary packages and run `jupyter notebook` and look at the individual .ipynb files and run them.

The .ipynb files are set up such that it is split up where each individual algorithm runs on a particular dataset.

The following packages need to be installed:
- sklearn
- graphviz
- numpy
- matplotlib
- pygame
- gymnasium
- bettermdptools
- swig
